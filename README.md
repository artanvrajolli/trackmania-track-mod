# TrackHunter

A desktop application for browsing and downloading Trackmania maps from the Trackmania Exchange API.

## Features

- Browse Trackmania maps from TM Exchange
- Download maps directly to your game
- Filter maps by category, style, and difficulty
- Search maps by name or author
- View map details and previews

## Installation

### From Release
Download the latest release from the [Releases](https://github.com/artanvrajolli/trackmania-track-mod/releases) page:
- **Portable**: Run directly without installation
- **Setup**: Install to your system

### Build from Source

```bash
# Install dependencies
npm install

# Start in development mode
npm run start

# Build executable
npm run build
```

## Tech Stack

- **Electron** - Desktop application framework
- **Trackmania Exchange API** - Map data source
